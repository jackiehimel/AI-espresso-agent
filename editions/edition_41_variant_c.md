## Ollama just raised $65M to help devs run AI models locally

The open-source tool that lets you run LLMs on your own machine—no cloud required—now has 9 million users and 176,000 GitHub stars. Benchmark led the round. Ollama makes it dead simple to download and run models like Llama or Mistral locally, which is why nearly 17,000 developers have forked it.

*Local AI is going mainstream—developers want models they can run and customize without API calls.*
